# Математика Семпай — Project Context

## Проект
- **Математика Семпай** — веб-приложение для обучения таблице умножения
- Целевая аудитория: дети 9–10 лет
- Методология: визуальный стратегический подход (исследование в корне проекта)
- Название: «Математика Семпай»

## Стек
- Vanilla HTML + CSS + JavaScript (без фреймворков)
- Статический деплой (Netlify/Vercel)
- Данные зашиты в `data.js`
- Прогресс — localStorage

## Структура
- `index.html` — каркас приложения (экраны, навигация)
- `style.css` — аниме-стиль (розовые тона, скругления, анимации)
- `data.js` — педагогическое ядро (CURRICULUM: недели, стратегии, визуальные подсказки)
- `script.js` — логика (навигация, уроки, визуализации, прогресс, достижения)

## Педагогические принципы (из исследования)
- Стратегический подход > зубрёжка (8× эффективнее по Brendefur 2015)
- 3 визуальные модели: массив, числовая прямая, пересечение палочек
- Модель Брунера: энактивный → иконический → символический
- Без давления скоростью (снижает тревожность)
- Интервальное повторение для долгосрочного запоминания
- Тултипы: методическое объяснение при наведении на любой элемент

## Ключевые стратегии по множителям
- ×2: удвоение | ×3: ×2 + ещё раз | ×4: двойное удвоение
- ×5: счёт пятёрками | ×6: ×5 + число | ×7: ×5 + ×2
- ×8: ×4 × 2 | ×9: ×10 − число (или пальцы) | ×10: приписать 0
