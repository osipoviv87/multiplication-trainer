# Матэ Додзё — Project Context

## Проект
- **Матэ Додзё** — веб-приложение для обучения математике
- Целевая аудитория: дети 9–10 лет (3 класс РФ)
- Методология: визуальный стратегический подход + передовые методики (Сингапур CPA, Япония, Китай)
- Тематика: тёплый светлый аниме/K-pop стиль (Ghibli/shoujo)

## Стек
- Vanilla HTML + CSS + JavaScript (без фреймворков)
- Статический деплой (GitHub Pages)
- Данные зашиты в `data.js`
- Прогресс — localStorage (ключ `mateDojoSave`)

## Структура
- `index.html` — каркас приложения (экраны, навигация)
- `style.css` — тёплый аниме-стиль (индиго/розовый/янтарь, мягкие тени, плавные анимации)
- `data.js` — педагогическое ядро (CURRICULUM: 3 раздела, 24 главы, теория, стратегии)
- `script.js` — логика (навигация, уроки, визуализации, прогресс, достижения, SVG-рендеры)

## Разделы и главы
- **Математика** (17 глав): сложение, вычитание, умножение ×2-×10, дроби (4 типа + равные + смешанные), деление, площадь, периметр, деление с остатком
- **Геометрия** (2 главы): виды углов, единицы длины
- **Физика** (5 глав): скорость, сила тяжести, рычаги, свет/тени, плотность

## Педагогические принципы
- Стратегический подход > зубрёжка (8× эффективнее по Brendefur 2015)
- Сингапур CPA: Concrete → Pictorial → Abstract
- Япония Lesson Study: hatsumon (крючок), kizuki (осознание)
- Китай Bianshi: вариация задач, несколько способов решения
- Финляндия: phenomenon-based learning (жизненные ситуации)
- Раскрывающийся блок теории в каждой главе (auto-open при первом визите)
- 15 программных SVG-иллюстраций для теории

## Тренировка — 4 режима
- **Карточки** (`flash`): flip-карты, spaced repetition
- **Марафон** (`marathon`): подсказка стратегии, вехи каждые 10
- **Босс** (`boss`): 10 HP, реплики, пошаговый разбор
- **Пойми** (`fluency`): объяснение ПОЧЕМУ метод работает

## Ключевые архитектурные паттерны
- `CURRICULUM.subjects.{math,geometry,physics}.weeks[]` с плоским `CURRICULUM.weeks`
- Новая операция = ветки в: `loadQuestion()`, `setVisual()`, `getExplanation()`, `check()`
- `loadTheory(weekId)` — заполняет блок теории из `week.theory`
- `renderTheorySVG(container, key, week)` — 15 SVG-рендереров
- `theoryVisited` в state — отслеживание первого визита

## Правила для Claude
- Все ответы на русском
- Простота важнее абстракций — не переусложнять
- UI дружелюбный для детей (крупные элементы, тёплые цвета, мягкие анимации)
- Никаких фреймворков — только HTML/CSS/JS
- Данные хранить в data.js, прогресс — в localStorage
